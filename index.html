This repository contains a Streamlit dashboard, not a static Vercel site.

Vercel cannot directly host Streamlit apps from a plain Python repo. To run the dashboard, use Streamlit Community Cloud or local execution.

Run locally

pip install -r requirements.txt
streamlit run app.py

Deploy on Streamlit Cloud

Push this repo to GitHub, then connect it at share.streamlit.io.

If you want the Vercel project to show a working page, this index page will now be served instead of 404.
